# Tapizados Nova

Web corporativa de Tapizados Nova, tapicería artesanal en Rubí (Barcelona), especializada en tapizado de sofás, sillas, cabeceros y mobiliario a medida.

## Tecnologías

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase

## Requisitos

- Node.js 18 o superior
- npm

## Instalación

```bash
npm install
```

## Desarrollo local

```bash
npm run dev
```

La web se abrirá normalmente en `http://localhost:5173`.

## Compilar para producción

```bash
npm run build
```

## Vista previa de producción

```bash
npm run preview
```

## Scripts disponibles

- `npm run dev`: inicia el servidor de desarrollo.
- `npm run build`: genera la versión de producción.
- `npm run lint`: revisa el código con ESLint.
- `npm run test`: ejecuta los tests.
- `npm run preview`: previsualiza la versión compilada.
